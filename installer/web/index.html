PayFin MCP — установка

Подключает 4 MCP-сервера в Claude Code. Без Docker. 2 минуты.

1. Установи

Открой PowerShell и выполни одну команду:

irm https://payfin-installer.up.railway.app/setup.ps1 | iex

На Mac / Linux / WSL: curl -fsSL https://payfin-installer.up.railway.app/setup.sh | bash

2. Перед запуском собери 4 токена

KB_TOKEN

Формат pfk_xxx. Получить у Sardorbek в Telegram.

GITLAB_TOKEN

  1. Зайди на git.ipoint.uz (нужен VPN)
  2. Правый верхний угол → Edit profile
  3. В меню слева: Access Tokens
  4. Жми Add new token. Scopes: read_api, read_repository
  5. Create → скопируй токен (покажется один раз)

CONFLUENCE_PAT

  1. Зайди на wiki.ipoint.uz
  2. Клик по аватару (правый верх) → Profile
  3. Personal Access Tokens → Create token
  4. Скопируй токен

JIRA_PAT

  1. Зайди на jira.ipoint.uz
  2. Аватар → Profile
  3. Personal Access Tokens → Create token
  4. Скопируй токен

3. Дополнительно выбери роль

Скрипт спросит AGENT_ROLE. Выбери одну:

РольДля кого
pmProduct Manager
analystБизнес-аналитик
developerРазработчик (по умолчанию)
tech-leadТех-лид (полный доступ)
qaQA-инженер

4. Перезапусти Claude Code

Закрой и открой заново. Проверь что MCP подключены:

claude mcp list

В выводе должны быть 4 сервера: payfin-kb, payfin-code, gitlab, atlassian.
